Descargador Helix (CTHD.exe)

La joya de la corona. Esta herramienta te permite explorar y descargar los miles de presets de Line 6 CustomTone con una interfaz fluida, moderna y potente.

Funcionalidades Principales:

  • Primera Ejecución Segura: El programa protege su configuración inicial con un velo oscuro hasta que establezcas y confirmes tu Carpeta de Salida.
  • Búsqueda Avanzada: Filtra por nombre, estilo, banda o autor. Usa el modo "Búsqueda AND" para combinar términos con precisión quirúrgica.
  • Filtro de Fecha: ¿Buscas lo más nuevo? Establece un rango de fechas "Desde" y "Hasta" para encontrar exactamente lo que necesitas.
  • Gestión de Descargas: Descarga múltiples presets simultáneamente. Si alguno falla, podrás reintentarlo con un solo clic.
  • Filtro Local: Una vez obtenidos los resultados, usa el buscador rápido 🔎 para filtrar instantáneamente entre los cientos de presets encontrados.
  • Menú Contextual: Clic derecho sobre cualquier preset para descargarlo o verlo directamente en la web de CustomTone.

Gestor de BD - CTHD Manager (Integrado & CTHD_Manager.exe)

El aliado perfecto para organizar tu colección local. Puedes lanzarlo desde el programa principal o desde el ejecutable que se genera en tu carpeta de descargas de forma independiente.

Funcionalidades del Gestor:

  • Auto-Carga Inteligente: Al iniciar, el gestor ya lista inmediatamente toda tu colección de presets para que los veas al natural. (También buscando por .).
  • Ver en Explorador: La columna interactiva incluye en cada fila un botón mágico Ver en explorador. Te llevará directamente seleccionándote en Windows el fichero `hlx`.
  • Buscador Local: Encuentra rápidamente por cualquier letra del nombre o banda cualquier preset ya bajado.
  • Copiar a (Standalone): En la aplicación independiente, pulsa un preset e instantáneamente te propondrá realizar un backup (Copia) a cualquier otra carpeta o pendrive.

Consejos de Uso Experto

Para exprimir al máximo tus herramientas:

  • Atajos de Teclado: En el descargador principal, puedes usar Shift o Ctrl para seleccionar rangos o presets individuales en la tabla, igual que en el explorador de Windows.
  • Búsqueda por Punto (.): Si dejas el término de búsqueda vacío o pones un punto ., el programa intentará traer los resultados más recientes disponibles.
  • Organización Local: Mantén el ejecutable CTHD_Manager.exe en la misma carpeta que tus presets para tener siempre acceso rápido a tu biblioteca, incluso sin internet.
  • Actualización de Manual: Si ves cambios en la aplicación, ejecuta el script de generación para tener siempre la última ayuda disponible.

Helix Downloader (CTHD.exe)

The crown jewel. This tool allows you to explore and download thousands of presets from Line 6 CustomTone with a fluid, modern, and powerful interface.

Main Features:

  • Safe First Run: The program protects its initial configuration with a dark overlay until you set and confirm your Output Folder.
  • Advanced Search: Filter by name, style, band, or author. Use the "AND Search" mode to combine terms with surgical precision.
  • Date Filter: Looking for the newest tones? Set a "From" and "To" date range to find exactly what you need.
  • Download Management: Download multiple presets simultaneously. If any fail, you can retry them with a single click.
  • Local Filter: Once results are obtained, use the quick search 🔎 to instantly filter through hundreds of found presets.
  • Context Menu: Right-click any preset to download it or view it directly on the CustomTone website.

DB Manager - CTHD Manager (Built-in & CTHD_Manager.exe)

The perfect ally to organize your local collection. You can launch it from the main program or from the standalone executable generated in your downloads folder.

Manager Features:

  • Smart Auto-Load: Upon startup, the manager immediately lists your entire preset collection naturally. (Also searchable using .).
  • View in Explorer: The interactive column includes a magic View in explorer button on each row. It will take you directly to the selected `hlx` file in Windows.
  • Local Search: Quickly find any downloaded preset by typing any letter of the name or band.
  • Copy to (Standalone): In the standalone app, tap a preset and it will instantly prompt you to create a backup (Copy) to any other folder or USB drive.

Expert Usage Tips

To get the most out of your tools:

  • Keyboard Shortcuts: In the main downloader, use Shift or Ctrl to select ranges or individual presets in the table, just like in Windows Explorer.
  • Dot Search (.): If you leave the search term empty or type a dot ., the program will attempt to fetch the latest available results.
  • Local Organization: Keep the `CTHD_Manager.exe` executable in the same folder as your presets to always have quick access to your library, even without internet.
  • Manual Updates: If you see changes in the application, run the generation script to always have the latest help documentation available.
